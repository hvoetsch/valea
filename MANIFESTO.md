# Valea Manifesto

Valea exists to make systems programming legible to both humans and autonomous agents.

## Principles

1. **Strict over clever**
   - one obvious syntax
   - explicit types
   - explicit errors

2. **Deterministic over magical**
   - deterministic project layout
   - stable diagnostics with fixed error codes
   - canonical formatting

3. **Repairable by default**
   - structured JSON diagnostics
   - stable AST export in JSON
   - no hidden control flow

4. **Systems-first**
   - static typing
   - compiled language model
   - no GC in core model
   - deterministic destruction (long-term language goal)

5. **Disciplined surface area**
   - small standard library
   - no exceptions
   - no macros
   - no implicit conversions

Valea is not trying to be all things. It is trying to be a dependable substrate for human and AI co-development.
